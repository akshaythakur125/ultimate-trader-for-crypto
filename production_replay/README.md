# production_replay — Dry-Forward Operator Layer

This directory contains the production-grade operator system for
running daily dry-forward validations. All files are safety-checked
and locked — do not modify without re-running Phase 5-7 validation.

## Files

### operator.py
**Single entry point for daily use.** Runs safety lock, launch check,
dry-forward, and evidence tracker in sequence. Generates 3 reports:
dry_forward_report.json, dry_forward_report.txt, operator_summary.txt.
Blocks immediately if any safety gate fails.

### evidence_tracker.py
Tracks evidence for paper/live unlock. Reads dry_forward_report.json
and maintains a run log in deploy_results/.evidence_tracker.json.
Reports whether each unlock gate passes or fails:
- Gate A: >= 100 trades
- Gate B: >= 30 calendar days
- PF >= 1.5, WR >= 35%, DD < 12.0R, EV > 0, kill not triggered

### safety_lock.py
Verifies all safety locks are engaged:
1. live_trading = false in config_locked.yaml
2. paper_trading = false in config_locked.yaml
3. dry_run = true in config_locked.yaml
4. eligible_for_live_trading = false in validation_gate.py
5. No API or order-execution imports in production_replay/

### launch_check.py
8-gate deployment blocker. Called by operator.py before any
dry-forward run. Gates:
1. live_trading_disabled
2. paper_trading_disabled
3. dry_run_enabled
4. no_blocked_configs
5. report_exists
6. allowed_configs_valid
7. git_tree_clean
8. bias_audit_passed

### run_dry_forward.py
Core validation engine. Runs 30/30/30 causal walk-forward with
regime gate for each allowed config. Uses consecutive_loss_cap(6)
risk control. No API calls, no order execution.

### config_locked.yaml
Single source of truth for all production configuration. Hard-locks:
- live_trading: false
- paper_trading: false
- dry_run: true
- allowed_configs (3 configs)
- blocked_configs (5 configs)
- stop_method: atr14_20
- entry_method: immediate
- regime_gate: enabled
- risk_control: consec_loss_6

**Do not edit this file manually.** Changes must go through the full
validation pipeline (Phase 5-7).
